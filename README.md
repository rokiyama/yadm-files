# yadm-files

## Setup

### macOS, Linux (amd64)

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install yadm
yadm clone --bootstrap https://github.com/rokiyama/yadm-files.git
```

### Ubuntu (arm64)

```sh
apt update && apt install -y yadm
yadm clone --bootstrap https://github.com/rokiyama/yadm-files.git
```
